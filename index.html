HTML

  • HTML is a computer language (code) use for content structure on a webpage. See it as the building frame of a house for a website. It is composed of multiple elements such as Head (Data defining our webpage); Body (contains all the content related to the HTML document, i.e: headings, paragraphs, hyperlinks, lists, images).
  • The head element contains information about the webage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is composed of selector and property which defines the rules the elements/attributes will follow.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • "*" selector is used to apply a rule to all elements visible on the webpage.
  • Text-align is used for positionning.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: adds new files in your directory.
  • git commit -m branche-name: commit your changes to your working directory with an associated message.
  • git push origin feature/branche-name: push the local branch to Github.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a data structure consisting of a collection of elements (values or variables), each identified by at least one array index or key.
  • A "For" Loop is used to repeat a specific block of code a known number of times.
  • A function is simply a block of code that you can use over and over again, rather than writing it out multiple times.
